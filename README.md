# Vite-Vue2-demo
开发环境vite，webpack打包。

- scss变量注入
- alias
- cdn引入，配置resolve externals
- html模板语法

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
# vite启动
npm run dev
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
